JupyterQuiz Tutorial for Educators

Learn how to embed interactive assessments into your Jupyter notebooks!

Programming Skill: Beginner
Primary Course: Instructor Tutorial
Format: Multi-Part Materials Module
Authors: Dr. Jessica A. Nash
Estimated Time: 30-45 minutes
python molecular-dynamics visualization quantum-chemistry

Overview

This is a lesson for educators that explains how to embed interactive assessments into Jupyter notebooks using the JupyterQuiz package. It goes over basics of the package and provides an engineered prompt so that large language models like ChatGPT can be used to format instructor questions.

Scientific Learning Objectives

  • None

Cyberinfrastructure Learning Objectives

  • Use JupyterQuiz to add interactive assessments to notebooks

Platform Requirements

Google Colab (Recommended)

No installation required. All dependencies are installed automatically when you run the notebooks.

  • Runs entirely in your web browser
  • Free Google account required
  • All required packages installed inline
  • Works on any device with internet
  • GPU acceleration available for computational tasks

ChemCompute

This module is compatible with ChemCompute - a specialized platform for computational chemistry education.

  • Access using your institution account
  • Pre-configured chemistry software environment
  • No personal setup required
  • Includes computational molecular science packages (Psi4, OpenMM)

Local Installation

For advanced users who prefer local development environments.

  • Requires conda package manager
